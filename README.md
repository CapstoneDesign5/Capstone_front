# 재가 요양 서비스 대상자 스마트 약통

## 웹
_사용자, 관리자, 약품, 약 복용 시간 관리_

### 🔎 주요 기능
1. 사용자, 약품 등록 및 수정·삭제
2. 약 복용 시간 입력
- 버튼 클릭시 전체 약통 잠금 해제 기능 추가
- 입력한 복용 시간은 DB에 저장되고 해당 시간이 되면 자동으로 약통의 잠금이 해제됨 (-잠금 해제 기능은 ESP32 코드 참고-)
3. 관리자 정보 변경
